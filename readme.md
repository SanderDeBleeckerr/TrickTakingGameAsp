# Tricktacking game simulation

This clingo program simulates the playing of the LOTR trick taking game.
Any reference to the copyrighted names are purely for educational use.

## Running the simulation

You might want to check the [clingo website](https://potassco.org/clingo/) first.

```bash
conda create -n potassco -c conda
activate potassco
clingo simple.lp 
```
